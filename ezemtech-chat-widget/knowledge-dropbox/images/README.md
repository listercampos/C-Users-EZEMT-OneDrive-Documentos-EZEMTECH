# Imagenes para referencia

Coloca aqui capturas, diagramas o imagenes que quieras usar como referencia tecnica.

El widget no interpreta imagenes directamente. Si una imagen contiene pasos, errores o instrucciones importantes, transcribelos a:

```text
../processed/local-knowledge.csv
```

Formato recomendado:

```csv
language,keywords,response,source
es,"error pantalla azul","Si aparece pantalla azul, anota el codigo de error y revisa drivers recientes.","captura-bsod.png"
```

No publiques imagenes con datos privados de clientes, numeros de serie sensibles, contrasenas o codigos de acceso.
