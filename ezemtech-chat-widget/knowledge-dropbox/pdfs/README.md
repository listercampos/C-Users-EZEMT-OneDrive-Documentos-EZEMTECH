# PDFs para conocimiento tecnico

Coloca aqui manuales, guias, procedimientos o PDFs que quieras convertir en conocimiento del agente.

Por seguridad y rendimiento, el widget no lee PDFs directamente desde esta carpeta. Extrae o resume la informacion importante y copiala en:

```text
../processed/local-knowledge.csv
```

Formato recomendado:

```csv
language,keywords,response,source
es,"manual drone dji","Para calibrar el drone, sigue estos pasos...","manual-dji.pdf"
```

No guardes contrasenas, datos privados de clientes ni documentos confidenciales en archivos publicos.
