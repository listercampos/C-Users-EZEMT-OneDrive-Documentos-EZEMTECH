## Instrucciones para nuevas actualizaciones

Para agregar otra actualizacion:

1. Crea un archivo nuevo en esta carpeta, por ejemplo `actualizacion 3.md`.
2. Escribe reglas, productos, procedimientos o respuestas autorizadas.
3. Agrega el archivo en `index.json` dentro de `updates`.
4. Sube el cambio a GitHub o al hosting.
5. Recarga la app para que cargue el nuevo conocimiento.

No pongas secretos, contrasenas ni datos privados de clientes en estos archivos.
