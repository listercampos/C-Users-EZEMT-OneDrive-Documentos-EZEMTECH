# Actualizaciones maestras EZEMTECH

Esta carpeta es para que el usuario maestro agregue informacion que el tecnico virtual debe aprender o usar como politica interna.

## Archivo inicial

```text
actualizacion 2.md
```

## Como agregar otra actualizacion

1. Crea un archivo nuevo en esta carpeta, por ejemplo:

```text
actualizacion 3.md
```

2. Escribe reglas, servicios, procedimientos o respuestas autorizadas.
3. Abre `index.json`.
4. Agrega una nueva entrada dentro de `updates`.
5. Sube los cambios al hosting o GitHub.
6. Recarga la app.

## Seguridad

No pongas aqui contrasenas, API keys, datos privados de clientes, tarjetas, SSN ni codigos de acceso.
